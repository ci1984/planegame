# 🚀 小飞机游戏

一个简单但功能完整的跨平台飞机射击游戏，使用HTML5 Canvas开发，支持PC端和移动端。

## 🎮 游戏特性

### 核心功能
- ✅ **响应式设计** - 完美适配各种屏幕尺寸
- ✅ **双输入控制系统** - PC端键盘 + 移动端触屏
- ✅ **完整游戏机制** - 玩家飞机、子弹系统、敌机生成
- ✅ **碰撞检测** - 精确的物理碰撞检测
- ✅ **游戏状态管理** - 得分、生命值、最高分记录
- ✅ **视觉效果** - 粒子爆炸、星空背景、发光特效

### 控制方式
#### PC端
- **移动**: 方向键 或 WASD
- **射击**: 空格键
- **暂停**: ESC键

#### 移动端
- **移动**: 左侧虚拟摇杆
- **射击**: 右侧红色射击按钮
- **暂停**: 右上角暂停按钮

## 🎯 游戏玩法

1. **目标**: 击落敌机获得分数
2. **生命**: 3条生命值
3. **失分**: 被敌机撞击或让敌机飞过屏幕底部
4. **难度**: 随着分数增加，敌机生成速度逐渐加快

## 🛠️ 技术栈

- **HTML5 Canvas** - 游戏渲染
- **原生JavaScript** - 游戏逻辑
- **CSS3** - 响应式样式和动画
- **LocalStorage** - 本地数据存储

## 📱 跨平台支持

- **桌面浏览器**: Chrome, Firefox, Safari, Edge
- **移动浏览器**: iOS Safari, Android Chrome
- **自动适配**: 检测设备类型并切换控制方式

## 🚀 快速开始

### 在线体验
访问 [GitHub Pages](https://ci1984.github.io/planegame/) 直接在浏览器中游玩

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/ci1984/planegame.git

# 进入项目目录
cd planegame

# 启动本地服务器
python3 -m http.server 8080

# 在浏览器中打开 http://localhost:8080
```
